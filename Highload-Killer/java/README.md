# HighloadKiller Java edition

### Required software

- Java Development Kit 1.7.0_67
- Apache Maven 3.2.3
- MySQL 5.6.20
 
### How to run
 
 1. Open Terminal.
 2. Go to folder `java/`
 3. Run command `mvn clean install`
 4. Run command `mvn jetty:run`
 5. Open browser window.
 6. Go to URL `http://localhost:8080`
